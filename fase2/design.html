Propósito del sitio web

El propósito de este sitio se crea con el fin de conocer, aprender y utilizar las etiquetas básicas de HTML, así como sus atributos y valores, todo con el fin de poder diseñar, crear e implementar recursos educativos digitales para dinamizar las estrategias de aprendizaje virtuales, ésto con la ayuda y la aplicación de diversas tecnologías que permite a los educadores crear diseños y guías de estudio para poder explicar los diferentes temas que serán motivo de estudio.

ETIQUETAS HTML

Etiquetas Básicas de HTML:

Primero vamos a saber y entender qué significa HTML.

Según Cordero, J. J. T. (2014), se define HMTL como el lenguaje estándar que se utiliza en la web para representar la información intercambiada por los usuarios en forma de documentos de hipertexto. Estos documentos contienen una serie de etiquetas que indican al navegador cómo interpretar y dar formato al texto plano. Según Cordero, J. J. T. (2014).

Etiquetas básicas de HTML:

Según Cordero, J. J. T. (2014) e Idevik, X. (2021)

Las etiquetas se dividen en tres partes; declaración inicial, de cabecera y de cuerpo.

Etiquetas de declaración inicial:

es la declaración inicial e indica a los navegadores que se trata de un documento HTML 5.

son las etiquetas de apertura y cierre de todo el documento (incluye cabecera y cuerpo).

Etiquetas de cabecera:

son etiquetas de cabecera que incluyen el título, la codificación de caracteres, etiquetas para los buscadores entre otros.

Etiquetas de cuerpo:

Indica la parte del cuerpo del contenido de un documento HTML. Es una etiqueta esencial para cualquier documento ya que indica donde empieza el contenido visible del documento.
