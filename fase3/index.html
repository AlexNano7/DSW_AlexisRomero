Inicio

Propósito del sitio web

El propósito de este sitio se crea con el fin de conocer, aprender y utilizar las etiquetas básicas de HTML, así como sus atributos y valores, todo con el fin de poder diseñar, crear e implementar recursos educativos digitales para dinamizar las estrategias de aprendizaje virtuales, ésto con la ayuda y la aplicación de diversas tecnologías que permite a los educadores crear diseños y guías de estudio para poder explicar los diferentes temas que serán motivo de estudio.


Introducción

Para poder diseñar y crear páginas web es importante conocer las etiquetas básicas que HTML nos ofrece junto con sus respectivos atributos y valores, ésto permite al diseñador de sitios web mejorar la experiencia hacia el usuario al hacerlo más agradable a la vista en cuanto al diseño del sitio, donde se deberá tener en cuenta la distribución de los espacios, el tipo y tamaños de letra a usar, la distribución correcta de las diferentes gamas de colores que expertos consideran agradables, entre otros factores a tener en cuenta para lograr así el mejor diseño.
